Introduction

Build or improve a web app focusing on accessibility standards (WCAG), ensuring keyboard navigation, screen reader support, and semantic HTML.

Tech Stack (optional)

  • Any frontend framework/library or Vanilla JS
  • ARIA attributes
  • Semantic HTML
  • Axe or Lighthouse for testing

Requirements

  • Ensure all interactive elements are keyboard accessible
  • Use proper ARIA roles and labels
  • Use semantic HTML tags (header, nav, main, footer, etc.)
  • Test accessibility with Lighthouse or Axe and fix issues
  • Provide skip navigation link
